⚠️ Windows Terminal Fix

Windows Terminal intercepts Ctrl+V by default. To paste images:

1. Open Terminal Settings → Actions
2. Change Paste from Ctrl+V to Ctrl+Shift+V
3. Restart terminal - now Ctrl+V works!
